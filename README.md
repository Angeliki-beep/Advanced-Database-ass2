# CRM Database Project

## Project Overview
This project implements a secure Customer Relationship Management (CRM) database solution.
The CRM acts as the Master of Contacts and supports integration with external systems.

## Learning Outcomes Covered
- LO1: Database design and implementation
- LO4: Database security threats and secure database implementation
- LO5: Evaluation of NoSQL and OODBMS solutions

## Technologies
- PostgreSQL / MySQL compatible SQL
- Relational Database Design
- Secure Access Control
